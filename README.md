## SetUp
1. install pyenv
```
brew install pyenv
```
2. install pipenv
```
pip install pipenv
```
3. designate python version 3.11
```
pipenv --python 3.11
```
4. activate pipenv
```
pipenv shell
```
5. install python package list refering to the Pipfile
```
pipenv install
```
6. run `main.py `
```
streamlit run main.py
```
